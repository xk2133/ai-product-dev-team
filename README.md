# AI 产品开发团队

> 基于 Claude Code 的 AI 多智能体产品开发框架 —— 从想法到上线，一站式产品开发解决方案。

## 📖 项目简介

**AI 产品开发团队** 是一个基于 Claude Code 的多智能体协作框架，通过主流程控制器 **Nina** 协调四个专业 AI sub-agent，实现完整的产品开发工作流程管理。你只需要提出想法，团队就能帮你完成从需求分析、产品设计、代码实现到后端部署的全流程。

## 🔄 开发工作流

```
需求分析 → PRD生成 → 设计策略 → 设计规范 → 技术方案 → 代码实现 → 后端部署
```

整个流程由 Nina 主导，每个关键节点需要用户确认后才能进入下一阶段：

| 阶段 | 指令 | 产出 |
|:-----|:----:|:-----|
| **需求收集与分析** | 自动启动 | 需求分析结果 |
| **PRD 生成** | `/PRD` | PRD.md 产品需求文档 |
| **设计偏好收集** | `/设计` | 设计策略 |
| **设计规范生成** | `/DRD` | DESIGN_SPEC.md 设计规范文档 |
| **技术方案规划** | `/开发` | 技术实现方案 |
| **代码实现** | `/开始` | 完整前端代码 |
| **部署配置** | `/部署` | 部署策略 |
| **部署方案生成** | `/上线` | 部署配置方案 |

## 🚀 快速开始

### 前置条件

- 安装 [Claude Code](https://claude.ai/code)（VS Code 扩展 或 CLI）
- 将本项目文件夹作为 Claude Code 的工作目录

### 使用方式

1. **克隆或下载本项目** 到本地目录

2. **在 Claude Code 中打开本项目文件夹**

3. **Nina 将自动启动**，向你打招呼并开始收集产品需求：
   ```
   嘿！👋 我是Nina，很高兴认识你！
   说吧，你想做什么？🚀
   ```

4. **跟随 Nina 的引导**，回答产品相关问题，逐步推进开发流程

5. **使用指令控制流程**：
   - 需求确认后输入 `/PRD` 生成产品需求文档
   - 输入 `/设计` 进入设计阶段
   - 输入 `/DRD` 生成设计规范
   - 输入 `/开发` 进入技术方案阶段
   - 输入 `/开始` 启动代码实现
   - 输入 `/部署` 进入部署配置阶段
   - 输入 `/上线` 生成部署方案

## ✨ 核心特性

- **全流程覆盖**：从想法收集到部署上线的完整产品开发链路
- **专业分工**：四个专业 sub-agent 各司其职，输出高质量的中间产物
- **用户主导**：每个关键节点需用户确认，确保方向始终正确
- **中文交互**：全流程中文对话，自然流畅
- **可扩展**：基于 Claude Code 的 sub-agent 架构，可灵活增加新的专业角色

## 🛠️ 技术栈

项目本身是基于 Claude Code 的 AI Agent 配置框架。在代码实现阶段，开发工程师默认使用：

- **前端**：HTML5 + CSS3 + JavaScript（MPA 多页应用架构）
- **部署**：Vercel（前端托管）+ Supabase（后端服务）
- **数据库**：PostgreSQL（Supabase）
- **认证**：Supabase Auth
